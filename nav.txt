npm run dev запуск проекта c tailwind css

tailwind.config.js настройки проекта

template html просто шаблон страницы header + footer

index.html главная

news.html стр с новостями 

contacts.html контакты + гугл карта

servise.html текст + галерея

media.html текст + цитаты + списки + таблицы + коментарии

events.html карточки Подій + календарь

style.css то что подключается ко всем html

input.css сюда пишем стили tailwind которые переносяться в style.css 

для навигации по input.css просто в поиск по файлу пишете название html файла. Весь адаптив, ховеры и прочее в input.css. В одном классе сразу прописуеться обычные стили, адаптив, ховеры, after и прочее. Также в класс можно прописать другой класс и стили возьмуться из класса который прописали внутрь класса. Например в классе quote, я написал класс body1C и из body1C беруться стили и передаються quote.

main.js тут весь js для всех страниц

comment.js тут логика коментариев

event-calendar.js тут логика календаря. Он на jquery.
